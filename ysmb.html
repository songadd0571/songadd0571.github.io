3 设计雨水流量计算

3.1 计算方法

设计雨水流量应按不同的分区，分别采用下式计算：

qv = (q × ψ × Fw) / 10000

式中：

  • qv—设计雨水流量 (L/s)，当屋面坡度大于 2.5% 或采用内檐沟集水时，设计雨水流量应乘以系数 1.5；
  • q—设计暴雨强度 [L/(s·hm²)]；
  • ψ—径流系数；
  • Fw—汇水面积 (m²)。

3.2 项目计算

本项目不同分区的数据如下表：

分区 汇水面积 Fw (m²) 径流系数 ψ 屋面坡度 (%) 是否采用内檐沟集水
分区 1 500 1.0 3.0 否
分区 2 800 1.0 2.0 否
分区 3 1000 1.0 2.0 是

不同分区的设计雨水流量 qv (L/s)

分区 1，qv = (614.751 × 1.0 × 500) / 10000 = 30.74 (L/s)。由于屋面坡度 3.0% > 2.5%，qv = 1.5 × 30.74 = 46.11 (L/s)。

分区 2，qv = (614.751 × 1.0 × 800) / 10000 = 49.18 (L/s)。由于屋面坡度 2.0% < 2.5%，且没有采用内檐沟集水，qv = 49.18 (L/s)。

分区 3，qv = (614.751 × 1.0 × 1000) / 10000 = 61.48 (L/s)。虽然屋面坡度 2.0% < 2.5%，但采用内檐沟集水，qv = 61.48 × 1.5 = 92.22 (L/s)。

4 管道设计

当采用 87 型雨水斗，为半有压流，DN100 的立管最大排水流量为 12 (L/s)。不同的分区所需的 DN100 雨水管数量 n ≥ qv/12，且 n 取整。

本项目雨水管道设计：

分区 1，n ≥ 46.11/12 = 3.84，所以 n 取 4；

分区 2，n ≥ 49.18/12 = 4.10，所以 n 取 5；

分区 3，n ≥ 92.22/12 = 7.69，所以 n 取 8。
