3 海绵计算

3.1 综合径流系数计算

综合径流系数ψz分为流量径流系数ψm和雨量径流系数ψc。

流量径流系数ψm，应根据下垫面类型按下式加权平均计算：

ψm = (∑ Fi × ψmi) / Fz

其中：

  • ψm —— 流量径流系数
  • Fi —— 汇水面上各类下垫面水平投影面积（m2）
  • ψmi —— 各类下垫面的流量径流系数
  • Fz —— 建设场地总面积（m2）

雨量径流系数ψc，应根据下垫面类型按下式加权平均计算：

ψc = (∑ Fi × ψci) / Fz

各类下垫面的雨量径流系数和流量径流系数如下：

下垫面类型 雨量径流系数ψci 流量径流系数ψmi
绿化屋面 0.30 - 0.40 0.40
硬质屋面、未铺石子的平屋面、沥青屋面 0.80 - 0.90 1.00
铺石子的平屋面 0.60 - 0.70 0.80
混凝土或沥青路面及广场 0.80 - 0.90 0.90 - 0.95
块石铺砌路面及广场 0.50 - 0.60 0.60

非铺砌路面 0.30 0.30

透水铺装 0.29 - 0.36 0.40

本项目的建设场地总面积：

Fz = 4,167.3 + 5,642.6 + 3,083.6 + 1,550.2 + 500 + 1,788.3 + 1,646.2 = 18,378.2 m2

本项目的流量径流系数：

ψm = (4,167.3×0.4 + 5,642.6×1 + 3,083.6×0.9 + 1,550.2×0.25 + 500×1.0 + 1,788.3×0.25 + 1,646.2×0.4) / 18,378.2 = 0.657

本项目的雨量径流系数：

ψc = (4,167.3×0.35 + 5,642.6×0.8 + 3,083.6×0.8 + 1,550.2×0.15 + 1,788.3×0.15 + 1,646.2×0.29) / 18,378.2 = 0.566

判断ψc是否满足控制指标中的要求：

本工程ψc=0.566，满足ψc≤0.6的要求。

3.2 年径流总量控制率核算

年径流总量控制率75%对应的日降雨量hp为26.4mm。

雨水通过自然和人工化的入渗、滞蓄、调蓄和收集回用，场地内累计一年需要控制的雨水总量V1为：

V1 = hp × Fz = 26.4 × 18,378.2 / 1,000 = 485.18 m3

其中，径流系数对应的年径流控制雨水量V2：

V2 = (1 - ψc) × hp × Fz = (1 - 0.566) × 26.4 × 18,378.2 / 1,000 = 210.57 m3

水面的调节水深为溢流水位和正常水位的差值，如果没有指定数值，按0.05m考虑。水面的控制雨水量Vs按面积投影面积Fi×0.05。

本项目水面的控制雨水量Vs = 500 × 0.05 = 25 m3

尚需控制的年径流雨水量：V3 = V1 - V2 - Vs = 485.18 - 210.57 - 25 = 249.6 m3

3.3 LID设施计算

设施一：设置下凹绿地，面积占比按40%计算：

下凹绿地面积 = 40% × (1,550.2 + 1,788.3) = 1,335.4 m2

拟将绿地面积1,335.4m2建成下凹式绿地（低于路面0.20m，蓄水高度0.15m），下凹式绿地受纳雨水容积：V4 = 1,335.4 × 0.15 = 200 m3

设施二：设置蓄水池，有效容积不小于V5 = 249.6 - 200 = 49.6 m3，可以采用雨水回用水池兼做调蓄功能。

结论：设置1,335.4m2下凹绿地，以及有效容积不小于49.6m3的蓄水池，即能满足年径流总量控制率不小于75%的要求。

3.4 面源污染消减率（以SS计）ηss计算

面源污染削减率（以SS计）ηss采用不同LID设施对应的污染物去除率ηi（%）按其服务面积进行加权平均进行计算。

ηss = (∑ FWi × ηi) / FWz

人工土壤渗滤 75 - 95

本项目的LID设施包括透水铺装、绿化屋面、蓄水池和绿地。

透水铺装、绿化屋面和绿地的服务面积FWi按对应的投影面积考虑。

蓄水池的服务面积FWi包括硬质屋面、混凝土或沥青路面及广场等的投影面积之和。

面源污染削减率（以SS计）：

ηss = [1,646.2×80% + 4,167.3×75% + (5,642.6 + 3,083.6)×80% + 1,550.2×60%] / (1,646.2 + 4,167.3 + 5,642.6 + 3,083.6 + 1,550.2) = 77%

ηss≥50% 满足控制指标要求。

3.5 透水铺装比例核算

透水铺装占路面及广场的比例 = 1,646.2 / (3,083.6 + 1,646.2) = 34.8%

满足≥25%的要求。

4 结论

本项目海绵城市设计满足以下控制指标要求：

  • 雨量径流系数ψc=0.566 ≤ 0.60，满足要求
  • 年径流总量控制率达到75%的要求，通过设置1,335.4m2下凹绿地和49.6m3蓄水池实现
  • 面源污染消减率ηss=77% ≥ 50%，满足要求
  • 透水铺装比例34.8% ≥ 25%，满足要求
  • 下凹绿地占绿地比例40% ≥ 40%，满足要求

本项目海绵城市设计方案合理，各项指标均达到或超过控制要求。
